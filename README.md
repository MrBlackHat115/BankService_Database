# **What is this database?**
* A MySQL-based banking database system designed to manage clients, accounts, transactions, employees, departments, cards, and bank branches (franchises). Includes sample data, triggers for automatic account updates, and indexes for faster queries. Ideal for learning database design, relational modeling, and basic banking operations.
# **Update:**
* Added transaction_logs table and a trigger to automatically update account balances and record transaction details (before/after balances) after each transaction.
